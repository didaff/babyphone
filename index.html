👶 Babyphone

Application locale, sans Internet ni serveur. Fonctionne entre deux téléphones Android connectés au même réseau (hotspot ou WiFi).

🍼Mode Bébé 👂Mode Parent ❓ Aide & procédure hotspot

Avant de commencer

1. Téléphone bébé : activer le point d'accès WiFi (hotspot) Android, brancher le téléphone sur secteur.
2. Téléphone parent : se connecter à ce hotspot (accepter « ce réseau n'a pas accès à Internet »).
3. Bébé → « Mode Bébé » → afficher le QR. Parent → « Mode Parent » → scanner le QR, puis montrer le QR réponse au bébé.
