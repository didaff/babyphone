Aide

📶 1. Créer le réseau (téléphone bébé)

Sur le téléphone qui restera près du bébé :
Paramètres → Réseau et Internet (ou Connexions) → Point d'accès et partage de connexion → Point d'accès Wi-Fi. Activez-le, notez le nom du réseau et le mot de passe. Le libellé exact varie selon la marque du téléphone (Samsung, Pixel, Xiaomi…) mais le chemin est toujours proche de celui-ci.
Branchez ensuite ce téléphone sur secteur : il va tourner en continu, écran allumé, caméra et micro actifs.

🍼 2. Lancer le Mode Bébé

Ouvrez l'app sur le téléphone bébé → Mode Bébé. Autorisez l'accès caméra/micro si demandé. Un QR code apparaît : c'est l'offre de connexion.

📱 3. Connecter le téléphone parent

Sur le téléphone parent : Wi-Fi → rejoindre le réseau créé à l'étape 1. Android va probablement afficher « Ce réseau n'a pas accès à Internet, se connecter quand même ? » → répondez oui (c'est normal et attendu, le babyphone n'a besoin d'aucun accès Internet).

👂 4. Lancer le Mode Parent et scanner

Ouvrez l'app sur le téléphone parent → Mode Parent. Autorisez l'accès caméra. Scannez le QR affiché par le téléphone bébé (l'offre). Un second QR (la réponse) s'affiche alors sur le téléphone parent : présentez-le à la caméra du téléphone bébé pour finaliser la connexion. Une fois connecté, l'image et le son du bébé s'affichent automatiquement.

📨 Envoyer l'app à un ami

Envoyez simplement l'URL de l'app (celle de cette page). Sur le téléphone qui la reçoit, ouvrir le lien puis menu du navigateur → « Ajouter à l'écran d'accueil » : l'app s'installe comme une vraie application et fonctionnera ensuite hors ligne.

⚠️ Limites à connaître

• Les deux écrans restent allumés (pas de fonctionnement en arrière-plan dans un navigateur) — le téléphone bébé doit rester branché sur secteur.
• Portée = portée du Wi-Fi (même bâtiment/appartement).
• Le premier chargement de l'app nécessite Internet une fois, sur chaque téléphone (installations et mises à jour uniquement — jamais pour l'usage courant).
• Si le téléphone bébé perd la connexion, le téléphone parent déclenche une alarme sonore forte après 10 secondes sans nouvelle : ne la désactivez pas tant que vous n'avez pas vérifié que tout va bien.
